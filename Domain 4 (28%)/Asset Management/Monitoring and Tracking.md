## Official Definition
The continuous process of maintaining an accurate and up-to-date inventory of all hardware and software assets within an organization.

## Key Mechanisms/Tools
*   **RFID (Radio Frequency Identification):** Using wireless tags for real-time location tracking of physical assets.
*   **Network Scanners:** Automatically discovering new devices that connect to the network.
*   **Agent-based Tracking:** Software installed on the asset that reports its status and location to a central server.
*   **CMDB (Configuration Management Database):** A central repository that stores information about all assets (CIs - Configuration Items) and their relationships.
*   **IP Address Management (IPAM):** Tracking which assets are using which IP addresses.

## Real-World Examples
*   An automated scan discovers an unauthorized "Shadow IT" server running in a branch office.
*   Using RFID tags in a hospital to track the location of expensive mobile medical equipment.

## Exam Tips
*   **Keywords:** CMDB, RFID, network discovery, inventory, automated scans.
*   **Scenario:** You can't secure what you don't know you have. **Inventory** is the foundation of security.
*   **Scenario:** Tracking assets across multiple sites? Use **Automated Discovery Tools**.

## Relationship to other concepts
*   **Vulnerability Management:** You must know an asset exists before you can scan it for vulnerabilities.
*   **Enterprise Security:** NAC (Network Access Control) relies on asset tracking to identify "known" vs. "unknown" devices.
*   **Incident Response:** Fast access to asset information helps responders understand the scope of an attack.
